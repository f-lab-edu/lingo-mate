### Lingo-Mate 프로젝트 소개
Lingo-Mate는 전세계 사람들과 함께 외국어 학습을 할 수 있는 플랫폼입니다. 

### 개발 목표
- 단어장, 질문 게시판을 기본 기능으로 하며 장기적으로 생성형 AI을 도입한 기능을 제공합니다.
- 성능적으로도 향상을 도모합니다.

### 개발 기간
2024/12/01 ~ ing

### 사용 기술
- Java 21 (LTS)
- Spring
  - Spring Boot
  - Spring Data JPA
- JJWT 0.12.3
- Database
  - MySQL
- Deployment
  - Docker
- Test
  - Mockito
  - Junit
